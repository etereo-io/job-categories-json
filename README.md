# JSON Job Categories

JSON containing job categories in spanish

See `job-categories.json` for more information.

Example: 

```
{
    "label": "Turismo y restauración",
    "value": "turismo-y-restauracion",
    "subcategories": [
      {
        "label": "Hostelería",
        "value": "hosteleria"
      },
      {
        "label": "Restauración",
        "value": "restauracion"
      },
      {
        "label": "Turismo",
        "value": "turismo"
      }
    ]
  }
```
